# Navod jak to vysvetlit zakum 8. tridy

Tento navod je psany jednoduse, aby se dal pouzit na prvni hodine Pythonu.
Soustredi se jen na zakladni pojmy.

## Cil hodiny (5 min)

- Zaci chapou, ze program je sada prikazu v poradi.
- Zaci umi spustit skript `kalkulacka.py`.
- Zaci vidi, jak program bere vstup a vypisuje vystup.
- Zaci poznaji `if/elif` jako vyber moznosti.

## Postup hodiny (20-30 min)

1) Spusteni
   - Ukaz, jak spustit `python kalkulacka.py`.
   - Nech jednoho zaka program spustit a zadat cisla.

2) Vysvetleni vstupu a vystupu
   - `print()` neco vypise na obrazovku.
   - `input()` se zepta uzivatele a ceka na odpoved.
   - Odpoved je vzdy text (retezec).

3) Prevod textu na cislo
   - `float("12.5")` udela z textu cislo 12.5.
   - Bez prevodu by neslo pocitat.

4) Vyber operace
   - `if/elif` znamena: "kdyz plati tohle, udelej tohle".
   - Ukaz kazdou vetvicku pro +, -, *, /.

5) Deleni nulou
   - Vysvetli, proc nejde delit nulou.
   - Ukaz, ze program to hlida.

## Otazky pro zaky

- Co se stane, kdyz do cisla napiseme pismeno?
- Co se stane, kdyz misto + zadame x?
- Proc je potreba `float()`?

## Jednoduche ukoly navic (10 min)

1) Uprav program tak, aby umel i mocniny `**`.
2) Vypisuj vysledek jako cele cislo, kdyz vysledek nema desetinnou cast.
3) Pridej opakovani, aby kalkulacka bezela porad dokola.

## Kratke shrnuti (2 min)

- Program = prikazy v poradi.
- `input()` nacte text.
- `float()` udela cislo.
- `if/elif` vybira spravnou akci.
